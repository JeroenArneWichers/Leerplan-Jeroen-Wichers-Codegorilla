Vaardigheden in het werkterrein

Veel van de vaardigheden die in het werkterrein gevraagd worden komen ook aan bod tijdens de bootcamp van code gorilla. Een greep uit de mogelijke openstaande functies leert ons veel. Functies zoals Software ontwikkelaar voor DeGraaf in Enschede hechten veel waarde aan technische kennis. "Je hebt bij de ontwikkeling onder andere te maken met C, PHP, Javascript, HTML, CSS en shell scripting" aldus DeGraaf. Veelgevraagde competenties komen vaak overeen. Zo wordt door Victa "Kennis van o.a. HTML5, CSS3, PHP en Javascript (jquery)."gevraagd voor de functie van Graphical Engineer / Front End Web Developer. Als je aan de slag wilt als junior developer (php) voor Adwise dan wordt "het programmeren in PHP,HTML,JS en MYSQL" gevraagd.

Dat niet enkel maar technische kennis gevraagd wordt komt ook snel naar voren bij het bestuderen van de openstaande vacatures. Zo verlangt DeGraaf dat je een "echte (scrum)teamspeler bent." Ook melden zij "Wij maken gebruik van de Atlassian tool chain en gaan er vanuit dat je bekend met versiecontrolesystemen." Voor de overige organisaties is het niet veel anders. Adwise ziet graag dat je als werknemer "een leergierige, gedreven en samenwerkingsgerichte instelling" hebt.
